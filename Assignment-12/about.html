About this static blog

This sample blogging website was created using only HTML and CSS. It demonstrates page linking, article structure, and visual consistency.

Purpose

The website is designed to show how a basic multi-page blog can be organized without JavaScript or backend features.

Author

Written as a practice concept by Aditya Kumar Shah for web programming exercises.

Pages Included

  • Home page with post summaries
  • Two separate static blog post pages
  • About page for context and site purpose
