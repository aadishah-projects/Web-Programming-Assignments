# Assignment 5: Interactive Quiz Application

## Overview
Quiz application with multiple choice questions stored as objects, progress tracking, and score calculation.

## Key Features
- Questions stored as objects with options and correct answers
- One question displayed at a time
- Multiple choice selection
- Progress bar showing completion
- Score tracking
- Answer review on completion
- Previous/Next navigation
- Responsive design

## JavaScript Concepts Used
- Objects: Quiz question structure
- Arrays: Questions array, user answers
- Array methods: forEach, filter
- Event handling
- DOM manipulation
- Conditional logic
- Calculation and percentage

## Implementation
- Quiz array with question objects
- Track user selections in array
- Calculate score by comparing answers
- Display progress visually
- Show answer review at end
