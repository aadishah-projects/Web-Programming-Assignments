# Assignment 4: Number Guessing Game with Score Tracking

## Overview
Interactive number guessing game where players guess a random number between 1-100 with a score that decreases with each wrong attempt.

## Key Features
- Random number generation (1-100)
- Score tracking (starts at 100, -1 per wrong guess)
- Feedback: "Too High", "Too Low", or "Correct"
- Track all previous guesses
- Play again functionality
- Keyboard support (Enter key)

## JavaScript Concepts Used
- Random number generation: `Math.random()`, `Math.floor()`
- Variables and operators
- Event handling (click, keypress)
- Conditionals (if-else)
- Array methods (push, map)
- DOM manipulation
- Game state management
- Score calculations

## Implementation Details
- Uses `Math.random()` for number generation
- Score decreases by 1 for each wrong guess
- Displays feedback based on comparison
- Shows all previous guesses with color coding
- Responsive design pattern matching other assignments
