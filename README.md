# Web Programming Assignment Collection

This project contains 10 Vanilla JavaScript assignments rebuilt with simpler mechanics and a consistent minimal interface while still matching the basic requirements from `LabQuestions.pdf`.

## Project Structure

- `Assignment-1` to `Assignment-10` contain the individual lab assignments
- `shared-assignment.css` provides the common minimal design system used across all assignments
- `LabQuestions.pdf` contains the source assignment requirements
- `assignments_report.tex` contains the combined LaTeX report source
- `README.md` provides the project overview

## Files Inside Each Assignment

Each assignment folder includes:

- `index.html` for the page structure
- `style.css` for assignment-specific styling layered on top of the shared minimal stylesheet
- `script.js` for the Vanilla JavaScript logic
- `Lab_Report.md` for the assignment report content

## Assignment Overview

1. `Assignment-1` - Smart Calculator with History  
   Performs basic arithmetic, stores the last 5 calculations, and supports clearing history.

2. `Assignment-2` - Student Marks and Grade Analyzer  
   Stores student details as objects, calculates total and percentage, assigns grades, and highlights pass or fail rows.

3. `Assignment-3` - Dynamic To-Do List with Status Filter  
   Adds, deletes, completes, and filters tasks while showing task counts.

4. `Assignment-4` - Number Guessing Game with Score Tracking  
   Generates a random number, gives high or low feedback, and decreases score after wrong guesses.

5. `Assignment-5` - Interactive Quiz Application  
   Shows one question at a time, records selected answers, and displays the final score at the end.

6. `Assignment-6` - Digital Clock and Countdown Timer  
   Displays a live digital clock and runs a countdown timer with an alert at zero.

7. `Assignment-7` - Dynamic Table Generator  
   Generates a table from row and column input, adds rows, deletes rows, and highlights even rows.

8. `Assignment-8` - Simple E-Commerce Cart System  
   Uses product objects, supports add and remove actions, updates quantities, and calculates totals dynamically.

9. `Assignment-9` - Form Validation System  
   Validates name, email, and password in real time, shows dynamic messages, and blocks invalid submission.

10. `Assignment-10` - Event-Driven Color and Theme Manager  
    Updates background, text, and button colors from a theme object and restores default values with reset.

## Design Notes

- The UI across assignments has been simplified to a minimal, consistent layout.
- All assignments use plain HTML, CSS, and Vanilla JavaScript only.
- The implementations were rewritten to stay close to the core lab requirements instead of adding extra interface complexity.

## How To Use

1. Open any assignment folder.
2. Launch its `index.html` file in a browser.
3. Interact with the page and verify the required behavior.
4. Refer to `LabQuestions.pdf` for the original requirements if needed.
